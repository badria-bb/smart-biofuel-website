# Smart Biofuel Website 🌱

This is an interactive educational website about biofuels.

## 🔍 Features
- Learn about biofuels and their types
- AI algae image classifier
- Biofuel calculator
- World map of biofuel production
- Fun quizzes and timeline
- DIY lab activities
